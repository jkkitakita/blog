1. 本記事のゴール

  1. HugoとGithub Pagesを使って、ブログを開設する
  2. 新規投稿時のシェルスクリプトを考える。

2. 前提

  1. 私は、新卒二年目（もうすぐ、三年目）
  2. インフラエンジニアをやった後、現在、スクラムマスター。
  3. なので、そこまで、技術的な知識はない。

3. はじめていきましょう！

  1. HugoとGithub Pagesを使って、ブログを開設する
