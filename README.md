## Getting Started

1. Install Hugo

    ```sh
    brew install hugo
    ```

1. Update a submodule

    ```sh
    git submodule update -i
    ```

1. Add Some Content

    ```shell
    hugo new post/YYYYMMDD.md
    ```

1. Start the Hugo server

    ```shell
    hugo server
    ```
